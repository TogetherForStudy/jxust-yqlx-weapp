# 江理一起来学微信小程序

这是一个基于Taro的微信小程序项目，为江西理工大学学生提供服务，欢迎各位开发者一起为小程序设计功能，贡献代码！

## 🚀 功能特性 | V1.0.6

### 📅 课表功能
- **智能课表展示**：采用7天×5节课的网格布局，清晰展示每日课程安排
- **周数切换**：支持按周查看课程，可快速跳转到当前周
- **课程详情**：点击课程卡片查看详细信息（教师、教室、时间等）
- **渐变色彩**：每门课程自动分配不同的渐变色，便于区分
- **班级切换**：支持搜索和切换不同班级的课表
- **当前周标识**：课程卡片会根据当前周显示不同透明度

### 🔍 发现页面
- **教师评价浏览**：查看热门教师评价和推荐
- **智能搜索**：支持按教师名称、课程名称搜索
- **功能入口**：集成写评价、我的评价等功能入口
- **评价分类**：支持推荐、避雷、中立三种评价态度

### 👤 个人中心
- **用户信息展示**：显示学号、班级、学院等个人信息
- **资料编辑**：支持修改个人信息
- **功能管理**：我的评价、意见反馈等功能入口
- **权限管理**：管理员用户显示额外的管理功能

### 🔐 认证系统
- **微信一键登录**：集成微信小程序登录API
- **Token管理**：自动处理认证token的存储和更新
- **权限控制**：需要认证的功能自动跳转到登录页面

### 📊 挂科率查询
- **课程搜索**：支持按课程名称搜索挂科率
- **随机浏览**：默认展示随机十条挂科率数据
- **等级标识**：根据挂科率高低显示不同颜色标识
- **详细信息**：显示开课单位、学期等课程信息

### 🏆 英雄榜
- **贡献者展示**：展示为一起来学做出贡献的用户
- **红色主题**：采用经典红色主题设计
- **统计信息**：显示总贡献人数和贡献类型

### 🗺️ 校区地图
- **多校区支持**：覆盖三江东园、三江西园、红旗校区、南昌校区
- **手风琴展示**：可展开收起的地图查看方式
- **图片预览**：支持点击放大查看详细地图
- **懒加载**：展开时才加载对应校区地图

### 📍 校区地址
- **详细地址**：提供各校区完整地址信息
- **邮编信息**：包含各校区邮政编码
- **一键复制**：支持快速复制地址和邮编信息

### 💬 群聊推荐
- **分类展示**：按不同类别展示学习群聊
- **群号复制**：一键复制群号便于加入
- **申请入群**：特殊群组支持直接申请加入

### 📅 校历查询
- **图片展示**：通过配置系统动态展示校历图片
- **高清预览**：支持点击查看高清校历
- **实时更新**：通过后台配置实时更新校历内容

### 🎓 成绩证明
- **图片展示**：展示成绩打印相关说明图片
- **快速查看**：便于学生了解成绩证明打印流程
- **预览功能**：支持点击预览详细信息

### 🌐 网页浏览
- **内嵌浏览器**：支持在小程序内浏览网页内容
- **动态标题**：根据页面内容动态设置导航栏标题
- **参数传递**：支持URL和标题参数传递

### ⚙️ 管理功能
- **配置管理**：管理员可管理系统各项配置参数
- **英雄榜管理**：管理员可添加、编辑、删除英雄榜成员
- **权限控制**：严格的管理员权限验证
- **数据筛选**：支持搜索和分类筛选功能

## 🛠️ 技术栈

- **框架**: Taro 4.x
- **前端**: Vue 3 + Composition API
- **状态管理**: Pinia
- **样式**: Weapp-Tailwind CSS
- **平台**: 微信小程序

## 🚀 开发指南

### 启动项目
```bash
# 安装依赖
npm install

# 微信小程序开发
npm run dev:weapp
```

### 构建项目
```bash
# 构建微信小程序
npm run build:weapp
```

## 🔧 配置说明

- 调整 `src/utils/constants.js` 中的学期配置
